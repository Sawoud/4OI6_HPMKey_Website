Summary

Due to the constant security breaches that around the world, to many major companies that are highly trusted by public, ensuring the safety of ones online identity is difficult. Especially due to that fact that many users create simple passwords that are very easy to crack by simple guessing algorithms, or when they do decide to create a complicated password, they end up just using that same password over many websites as that is the only random password they have committed to memory, but if only one of these websites is hacked, then their entire online identity is at risk. The Hardware Password Manager, The HPMKey, aims to solve this issue by storing the users information on a portable USB device that uses SHA256 encryption, the HPMKey also aims to make the process of generating random passwords very simple so that the user is encouraged to create a random password for every website they use.
